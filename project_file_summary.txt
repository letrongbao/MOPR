Project File Function Summary

Purpose
- This app is a rental/property management Android app.
- Vietnamese domain data keys and Firestore collection names are kept for backward compatibility unless a future migration explicitly changes them.

Core
- app/src/main/java/com/example/myapplication/core/MyApplication.java: application bootstrap and global initialization.
- app/src/main/java/com/example/myapplication/core/receiver/NetworkReceiver.java: listens for network state changes.
- app/src/main/java/com/example/myapplication/core/service/ImageUploadService.java: uploads images.
- app/src/main/java/com/example/myapplication/core/service/InvoiceReminderWorker.java: background invoice reminder notifications.
- app/src/main/java/com/example/myapplication/core/widget/InvoiceWidgetProvider.java: home-screen widget provider for invoice status.
- app/src/main/java/com/example/myapplication/core/util/ContractStatusHelper.java: contract status helper logic.
- app/src/main/java/com/example/myapplication/core/util/ContactHelper.java: contact-related helper utilities.
- app/src/main/java/com/example/myapplication/core/util/FinancePeriodUtil.java: month/year normalization and finance period helpers.
- app/src/main/java/com/example/myapplication/core/util/MoneyFormatter.java: currency and amount formatting.
- app/src/main/java/com/example/myapplication/core/util/NonScrollGridView.java: custom grid view that disables internal scrolling.
- app/src/main/java/com/example/myapplication/core/constants/InvoiceStatus.java: invoice state constants.
- app/src/main/java/com/example/myapplication/core/constants/RoomStatus.java: room state constants.
- app/src/main/java/com/example/myapplication/core/constants/TenantRoles.java: tenant role constants.
- app/src/main/java/com/example/myapplication/core/constants/TicketStatus.java: ticket state constants.
- app/src/main/java/com/example/myapplication/core/repository/AuditLogRepository.java: audit log persistence.
- app/src/main/java/com/example/myapplication/core/repository/domain/*Repository.java: data-layer repositories for invoices, expenses, houses, rooms, tenants, meter readings, rental history, tickets, and payments.
- app/src/main/java/com/example/myapplication/core/session/*: tenant session and invite/session state.
- app/src/main/java/com/example/myapplication/core/service/InvoiceReminderWorker.java: tracks unpaid invoices and sends reminders.
